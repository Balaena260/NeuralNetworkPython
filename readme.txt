НС - 0.1 / Нейронная сеть 0.1
Первая версия движка для обучения нейросетей общего назначения. 

Описание модулей проекта: 
    -ns0.1 сама нейронная сеть. Отвечает за feedForward. Рассчитывает loss. 
    Может скопировать себя. Изменяет веса. 
    -so.py / система обучения. Обучает нейросети с помощью ГА. Отвечает за: 
    создание поколений, прогон большого колличества нейросетей по батчу, 
    выбор лучшей нейронной сети.
    -storage_json.py библиотека для сохранения и загрузки нейросети в json файл.
